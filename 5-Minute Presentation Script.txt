5-Minute Presentation Script

AI-Powered Campus Navigation & Smart Help Desk System

Slide 1 — Title

Script

Good morning/afternoon.
Our project is AI-Powered Campus Navigation & Smart Help Desk System.

This system is designed to help users easily navigate large campuses using interactive maps and AI-based assistance.

The project was developed under the guidance of Mr. Himanshu Kaushik.

Slide 2 — Project Overview

Script

Large campuses often contain many buildings and departments spread across wide areas.

This makes navigation difficult, especially for new students and visitors.

Traditional solutions like signboards and help desks are limited and not always efficient.

Our system provides a web-based navigation platform with an AI chatbot that can guide users and provide campus information instantly.

Slide 3 — Objectives

Script

The main objective of this project is to create a centralized campus navigation system.

Key goals include:

Providing real-time location search

Integrating an AI chatbot for queries

Reducing dependency on physical help desks

Improving accessibility for campus users

Slide 4 — Key Features

Script

The system offers several core features.

Interactive campus map for location guidance

AI chatbot to answer queries

Search functionality for buildings and facilities

Web-based interface accessible from any device

These features make campus navigation simpler and faster.

Slide 5 — Project Use Cases

Script

The system can assist different users within a campus.

For example:

Students locating classrooms or departments

Visitors finding administrative offices

Users asking campus-related questions through the chatbot

The system can also be adapted for universities, hospitals, and corporate campuses.

Slide 6 — System Workflow

Script

The workflow of the system is simple.

The user accesses the web platform and submits a search or chatbot query.

The backend processes the request and retrieves data from the campus database and mapping service.

Finally, the system displays navigation routes or information to the user.

Slide 7 — Technologies Used

Script

The project is developed using modern web technologies.

Frontend: React.js, HTML, CSS, JavaScript

Backend: Node.js and Express.js

Database: MongoDB

Mapping Services: OpenStreetMap / Mapbox

AI: NLP-based chatbot integration

These technologies allow the system to be scalable and efficient.

Slide 8 — Individual Work Contribution

Script

The project development was divided into two main parts.

Frontend Development — Varun

User interface design

Interactive campus map integration

Search and navigation interface

Backend Development — Siddhant

API development using Node.js

Database management using MongoDB

AI chatbot and system logic integration

Slide 9 — Expected Results

Script

The system aims to provide accurate campus navigation and quick access to information.

It reduces reliance on physical help desks and improves accessibility for users.

Overall, it helps campuses move toward a smarter and more digital environment.

Slide 10 — Future Scope

Script

In the future, the system can be expanded with additional features such as:

Indoor navigation within buildings

Voice-based chatbot interaction

Mobile application development

Multilingual support

These improvements can make the system a complete smart campus solution.

Slide 11 — Thank You

Script

This concludes our presentation.

Thank you for your attention.
